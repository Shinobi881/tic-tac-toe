##Installation

*Before running you should have Webpack and Babel installed globally:*

`$ npm i -g webpack babel babel-core` 

*After completing the installation above. From your CLi:*

1. `$ npm i`
2. `$ npm start`
3. Then open a web browser to localhost:3000/
